# Organizador de Tareas

Programa de consola en Python que permite gestionar una lista de tareas: agregar, marcar como completadas y consultar las pendientes.

## ¿Qué hace?

- Agregar nuevas tareas a una lista
- Marcar tareas como completadas
- Ver las tareas pendientes
- Salir del programa

## Conceptos de Python practicados

- Estructuras de control: condicionales (`if`/`elif`/`else`)
- Bucles (`while`)
- Funciones con parámetros y retorno de valores
- Listas y su manipulación (`append`, `remove`)
- Entrada de datos por consola (`input`)

## Cómo ejecutarlo

1. Clona este repositorio o descarga el archivo `tareas.py`
2. Asegúrate de tener Python instalado
3. Desde la terminal, navega a la carpeta del proyecto
4. Ejecuta:

\`\`\`bash
python tareas.py
\`\`\`

5. Elige una opción del menú escribiendo el número correspondiente (1-4)
